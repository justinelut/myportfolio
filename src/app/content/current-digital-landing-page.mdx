- ## Increased user conversions
  Users visiting Current digital's website increased which inturn increased the user conversion renovation
  from when the site had been built with wordpress. this can be attributed to the following **Nextjs Features**

1.  Better initial load layout-shift management
2.  Smooth transition between pages
3.  No page reloads between pages which might increase server load from pages that fetch data from pocketbase.
