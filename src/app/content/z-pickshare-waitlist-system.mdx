# Pickshare – A Waitlist System for Handicap Betting

## Project Overview

As part of the build-up to Pickshare.co's launch, I developed a waitlist system designed to engage users before the official release. The system not only manages signups but also creates excitement and anticipation among potential users, setting the stage for a successful platform launch.
